# Dinosaur Game 🦖

A Chrome Dinosaur style endless runner game developed using Java and Java Swing.

## Features

- Dinosaur jumping mechanics
- Random cactus obstacles
- Collision detection system
- Score tracking
- Game over screen
- Restart game functionality
- Smooth animation and movement
- Keyboard controls

---

# Technologies Used

- Java
- Java Swing
- Java AWT
- Object-Oriented Programming (OOP)
